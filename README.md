# Professional Food Delivery POS System 🍕🍔

A comprehensive Point of Sale (POS) system designed for restaurants, cafes, and food delivery businesses. Built with a modern tech stack ensuring speed, reliability, and professional features.

## 🚀 Key Features

### 🛒 Point of Sale (POS)
*   **Dynamic Order Management**: Support for **Dine-In**, **Takeaway**, and **Delivery**.
*   **Real-time Calculations**: Automatic calculation of Subtotal, Tax (VAT), and Service Charges based on dynamic settings.
*   **Delivery Zone Management**: Integrated delivery zones with automated fee calculation.
*   **Table Management**: Visual table selection and status tracking (Available/Occupied).
*   **Order Parking**: Keep multiple Dine-In orders open simultaneously.

### 🍳 professional Kitchen & Operations
*   **Kitchen Printer Integration**: Automatic generation of thermal-style kitchen receipts with large order numbers for easy reading.
*   **Order Number Display**: High-visibility order number modal after creation to improve customer service.
*   **Item Customization**: Add notes to specific items for kitchen instructions.

### 📊 Management & Admin
*   **Menu Management**: Organize items by Main Groups, Sub Groups, and Categories.
*   **Uncategorized Support**: Handles items without specific categories gracefully.
*   **Order Tracking**: Complete order history with status updates (Pending, Preparing, Ready, Delivered, Cancelled).
*   **Revenue Analytics**: Real-time stats on daily sales, pending orders, and completed transactions.
*   **Platform Settings**: Control tax rates, service charges, and system-wide configurations.

### 🔐 Security & Multi-User
*   **Role-based Access**: Secure admin and staff levels.
*   **JWT Authentication**: Secure API communication.
*   **Shift Tracking**: Detailed shift logging for staff accountability.

## 🛠 Tech Stack
*   **Frontend**: React, TypeScript, Vite, Tailwind CSS (or Modern Vanilla CSS).
*   **Backend**: Node.js, Express.
*   **Database**: PostgreSQL.
*   **Utilities**: Axios, JWT, Dotenv.

## 📦 Professional UI/UX
*   **Dark Mode Support**: Fully responsive dark/light mode.
*   **RTL Support**: Optimized for Arabic and Right-to-Left languages.
*   **Modern Aesthetics**: Premium design with smooth gradients and micro-animations.
